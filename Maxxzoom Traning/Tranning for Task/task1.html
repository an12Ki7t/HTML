Project

Ag cafe Home page

The AG Cafe Website is a full-stack project with a pastel-themed UI, built using React + TailwindCSS
for the frontend and Node.js + Express for the backend. It features a clean homepage with sections for Drinks,
Snacks, and Desserts, allowing users to view
product images, prices, ingredients, delivery details, and payment options.

and parents.”
